# 第 1 章  计算机系统漫游
## 1.1 信息就是位 + 上下文

源程序实际上就是一个由值 0 和 1 组成的位 (又称为比特) 序列，8 个位被组织成一组，称为**字节**。

## 1.2 程序被其他程序翻译成不同的格式
每条 C 语句都必须被其他程序转化为一系列的低级机器语言指令。然后这些指令按照一种称为可执行目标程序的格式打好包，并以二进制磁盘文件的形式存放起来。目标程序也称为可执行目标文件。
