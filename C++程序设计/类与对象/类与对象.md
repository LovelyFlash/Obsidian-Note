# 类与对象

C++ 是面向对象的程序设计语言，具有 封装、继承、多态的特性，而其关键就是类。
类是创建对象的蓝图或模板。它定义了对象的属性（数据成员）和行为（成员函数）。通过类，我们可以将数据和操作数据的方法封装在一起，形成一个独立的单元。这有助于实现代码的模块化、重用性和可维护性。
