# 1 模版的概念

- 模板是 C++ 中实现<font color="#4f81bd">泛型编程</font>的核心机制，它允许编写与具体数据类型无关的通用代码。模板本质上是一种<font color="#4f81bd">代码生成工具</font>，编译器在编译时根据实际使用的数据类型自动生成相应的具体代码；
- 模板是对一族函数或类的描述，是参数化的函数和类；
- 模板是一种参数化的多态性工具，为各种逻辑功能相同而数据类型不同的程序提供一种代码共享的机制。模板不是以数据为参数，而是以它所使用的数据类型为参数；

**核心思想**
1. 类型参数化：
将数据类型作为参数传递给模板；
相同的算法逻辑可以应用于不同的数据类型；
实现 “一次编写，多处使用” 的目标。
2. 编译时多态：
模板在编译时实例化，而不是运行时；
编译器为每种使用的数据类型生成特定的代码版本；
没有运行时性能开销。
