# 变量模板
变量模版是 C++14 引入的一个特性，它允许你定义一个变量，这个变量本身是参数化的（即它的类型或值可以根据模板参数而变化）。  
在此之前，只有函数和类可以被模板化。变量模板扩展了模板的适用范围。  

**语法**  
 ```C++
 template <typename T>  
T variable_name = initial_value;  
  
// 或者带有非类型模板参数  
template <int N>  
int array_size = N;  
```

**作用和优点**  
1. 类型依赖的常量： 定义一个常量，其类型或值取决于模板参数；
2. 泛型配置： 提供泛型的配置值或默认值；
3. 简化代码： 避免为不同类型重复定义相似的常量或变量。  

**示例** 
```C++
 #include <iostream>  
#include <string>  
  
// 1. 定义一个变量模板：pi 的类型取决于模板参数 T  
template <typename T>  
constexpr T pi = T(3.1415926535897932385L);  
  
// 2. 定义一个变量模板：用于获取容器的默认容量  
template <typename Container>  
constexpr size_t default_capacity = 10;  
  
// 3. 定义一个变量模板：用于获取数组的默认值  
template <typename T>  
T default_value = T(); // 默认构造  
  
int main() {  
// 使用 pi 变量模板  
std::cout.precision(10);  
std::cout << "pi<double>: " << pi<double> << std::endl;  
std::cout << "pi<float>: " << pi<float> << std::endl;  
std::cout << "pi<long double>: " << pi<long double> << std::endl;  
  
// 使用 default_capacity 变量模板  
std::cout << "default_capacity<std::vector<int>>: " << default_capacity<std::vector<int>> << std::endl;  
std::cout << "default_capacity<std::string>: " << default_capacity<std::string> << std::endl;  
  
// 使用 default_value 变量模板  
int int_default = default_value<int>;  
std::string string_default = default_value<std::string>;  
std::cout << "default_value<int>: " << int_default << std::endl;  
std::cout << "default_value<std::string>: '" << string_default << "'" << std::endl;  
  
return 0;  
}
```
